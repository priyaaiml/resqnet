# ResQNet - Community Disaster Management System

Community-powered disaster response, built for the moments that matter.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express.js
- Database: MongoDB Atlas
- Real-time: Socket.io
- Auth: JWT
- Images: Cloudinary

## Features
- Disaster reporting with images
- Live disaster feed with filters
- Role based access (User, Volunteer, Admin)
- Admin dashboard with analytics
- Volunteer management system
- Real-time notifications
- Interactive disaster map
- Incident status tracking

## Setup

### Backend
cd server
npm install
npx nodemon server.js

### Frontend
cd client
npm install
npm run dev

## Environment Variables
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
CLOUDINARY_CLOUD_NAME=your_name
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret
